# MyLogger

**MyLogger** is a lightweight, reusable logging library for Java 17+.  
It provides simple static methods to log messages to the console with **color-coded levels** and timestamps.  

---

## Features

- Supports four logging levels: `INFO`, `WARN`, `ERROR`, `DEBUG`
- Automatic timestamping
- Color-coded console output for better readability
- Lightweight and easy to integrate

---

## Installation

Add the following dependency to your Maven project:

```xml
<dependency>
    <groupId>io.github.abduraxim-uz</groupId>
    <artifactId>mylogger</artifactId>
    <version>1.0.1</version>
</dependency>
